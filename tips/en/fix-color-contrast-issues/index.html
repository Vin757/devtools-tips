Fix low color contrast issues

With DevTools you can detect low color contrast issues which is great. But DevTools in Chrome, Polypane and Edge go one step further and help you fix these issues too! Domi tells us all about it.

In Polypane, these suggestions are made inline on the page itself when enabling the Contrast checker debug tool.
